游戏中的角色具有如下面板信息：

| 属性         | 描述                                                               |
| ------------ | ------------------------------------------------------------------ |
| 姓名         | 角色的名字                                                         |
| 悟性         | 基本三维之一，记为 $x$                                             |
| 根骨         | 基本三维之一，记为 $y$                                             |
| 体魄         | 基本三维之一，记为 $z$                                             |
| 武学素养     | 通过学习、修行功法提升的可成长属性，记为 $A$                       |
| 生命值       | 战斗时的HP，初始为 $100z$                                          |
| 内息量       | 战斗时的MP，通过修行获取                                           |
| 基础攻击力   | 不使用攻击武技时的攻击力，初始为 $3z$                              |
| 基础防御力   | 不使用防御武技时的防御力，初始为 $2z$                              |
| 最大内息输出 | 使用武技时最大消耗的内息百分比，初始为 $0.3y\%$                    |
| 内息输出     | 每次攻击时使用的内息百分比，若超过最大内息输出则限制为最大内息输出 | 
| 内功         | 当前修行的内功                                                     |
| 攻击武技     | 当前使用的攻击武技                                                 |
| 防御武技     | 当前使用的防御武技                                                 |
| 内功等级     | 内功等级                                                           |
| 攻击武技等级 | 攻击武技等级                                                       |
| 防御武技等级 | 防御武技等级                                                       |
| 威能         | 攻击武技属性                                                       |
| 守御         | 防御武技属性                                                       |
| 内息质量     | 内功属性                                                           |
| 出手速度     | 内功属性                                                           |
| 回气量       | 内功属性                                                           |
| 蓄力时间     | 攻击武技属性                                                       |
| 增伤         | 战斗输出增伤                                                       |
| 减伤         | 防御时减伤                                                         |
| 减伤上限     | 初始为50\%                                                         |
| 特性         | 人物天赋，带有战斗、修行时特效                                     |
| 战斗特效     | 由特性、武技、内功带来的战斗时特效                                 |

这一面板主要在修行与战斗时使用，其中基础三维和武学素养为修行属性，特性之外的为战斗属性，分别在 [[修行系统]] 和 [[战斗系统]] 中详述，此处只具体化特性。

特性可以理解为人物天赋，永久生效，可以通过开局挑选、修行、奇遇三种方法获得。特性除了名字和描述外，核心能力是为角色提供**词条**。在角色行动时，词条在特定时刻发挥作用，具体时刻以及对应时刻合法的效果如下：

- 开局
	- 增加/减少三维，可突破或不可突破上限
- 获得时
	- 增加/减少三维，可突破或不可突破上限
- 阅读功法时
	- 获得的武学素养增加/减少X%
- 修行内功时
	- 修行经验增加/减少X%
- 修行攻击武技时
	- 修行经验增加/减少X%
- 修行防御武技时
	- 修行经验增加/减少X%
- 内功升级时
	- 获得的内息量增加/减少X%
	- 获得的武学素养增加/减少X%
	- 增加/减少三维，可突破或不可突破上限
- 攻击武技升级时
	- 获得的武学素养增加/减少X%
	- 增加/减少三维，可突破或不可突破上限
- 防御武技升级时
	- 获得的武学素养增加/减少X%
	- 增加/减少三维，可突破或不可突破上限
- 转修时
	- 损失的内息量增加/减少X%

以下为战斗独有特效，其中具体X的计算可依据敌我双方战斗时面板进行计算：
- 战斗开始时
	- 生命值上限/生命值/内息量上限/内息量/基础攻击力/基础防御力/最大内息输出/内息输出/出手速度/回气量/蓄力时间/增伤/减伤/减伤上限增加/减少X或变为X
- 人物攻击时
	- 生命值/内息量/基础攻击力/基础防御力/增伤/减伤（临时）增加/减少X或变为X
	- 对方生命值/内息量/基础攻击力/基础防御力/增伤/减伤临时增加/减少X或变为X
- 人物防御时
	- 生命值/内息量/基础攻击力/基础防御力/增伤/减伤（临时）增加/减少X或变为X
	- 对方生命值/内息量/基础攻击力/基础防御力/增伤/减伤临时增加/减少X或变为X
- 人物攻击后（此时计算X可用的变量还包含此次攻击结果）
	- 生命值/内息量/基础攻击力/基础防御力/出手速度/蓄力时间/回气量/增伤/减伤/减伤上限增加/减少X或变为X
	- 对方生命值/内息量/基础攻击力/基础防御力/出手速度/蓄力时间/回气量/增伤/减伤/减伤上限增加/减少X或变为X
	- 额外发动输出为X的攻击
- 人物防御后
	- 生命值/内息量/基础攻击力/基础防御力/出手速度/蓄力时间/回气量/增伤/减伤/减伤上限增加/减少X或变为X
	- 对方生命值/内息量/基础攻击力/基础防御力/出手速度/蓄力时间/回气量/增伤/减伤/减伤上限增加/减少X或变为X
	- 额外发动输出为X的攻击
- 战斗回合结束后（一次攻防）
	- 生命值/内息量/基础攻击力/基础防御力/出手速度/蓄力时间/回气量/增伤/减伤/减伤上限增加/减少X或变为X

词条的触发可以包含条件，战斗外触发条件如下：
- 当修行的内功为XXX或内功类型为XXX时
- 当修行的攻击武技为XXX或武技类型为XXX时
- 当修行的防御武技为XXX或武技类型为XXX时
- 当具备特性XXX时
- 当悟性/根骨/体魄小于，小于等于，等于，大于，大于等于X时

战斗时触发条件如下：
- 当战斗时生命值/内息量小于，小于等于，等于，大于，大于等于X时
- 当战斗时你的对手的生命值/内息量小于，小于等于，等于，大于，大于等于X时
- 当战斗时你的对手修行的内功/攻击武技/防御武技为XXX或类型为XXX时
- 当攻击击破敌方内息防御时
- 当攻击未击破敌方内息防御时
- 当成功内息防御敌方攻击时
- 当未成功内息防御敌方攻击时

以上条件可用“与”“或”结合。

词条内部对触发次数进行计数，可限制词条的触发次数，每场战斗刷新。
