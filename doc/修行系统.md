角色局内成长依靠行动点，每两个剧情事件之间可使用X个行动点，此处X由剧情线决定。每个行动点可以用于修行或游历，游历为随机触发事件，具体见[[事件系统]]，而修行主要是内功、攻击武技、防御武技的修行，这三者统称功法，都具有5个境界。

内功具有如下结构：

| 属性     | 描述                                                                 |
| -------- | -------------------------------------------------------------------- |
| 名字     | 内功的名字                                                           |
| 描述     | 内功描述                                                             |
| 修行公式 | 基于角色三维和武学素养的公式，结果为消耗一个行动点所能获得的修行经验 |
| 等级     | 当前等级                                                             |
| 稀有度   | 内功稀有度，分5级                                                    |
| 类型     | 字符串，可自定义                                                     | 
| 境界     | 五个等级内功的具体属性                                               |

内功境界为单独的结构，内功中的境界列表依序为1-5级的效果，境界具有如下属性：

| 属性     | 描述                                   |
| -------- | -------------------------------------- |
| 等级     | 境界等级                               |
| 经验     | 升至当前等级需要的经验                 |
| 内息量   | 升至当前等级后获得的内息量             |
| 武学素养 | 升至当前等级后获得的武学素养           | 
| 内息质量 | 当前等级的内息质量                     |
| 出手速度 | 当前等级的出手速度                     |
| 回气量   | 当前等级的回气量，为最大内息量的百分比 |
| 词条     | 当前等级内功具备的词条列表             |

获取功法后，哪怕不修行，角色也会获得基于稀有度的武学素养，数值暂未定。

同一时间只能修行一门内功，转修会亏损内息，低稀有度到高稀有度的转修每高一个等级亏损15%的内息，反之不亏损但也不增加。即，如果修行稀有度为1的内功时内息量为1000，那么转修稀有度为5的内功时，内息量会变为400。转修后原内功等级保留，重新转修回来词条仍在，但内息量不会重复获得。转修不消耗行动点，但升到1级需要消耗行动点获取修行经验，因此内功结构中等级范围为0-5，还要包括当前经验条。

攻击武技的属性如下：

| 属性     | 描述                                                                 |
| -------- | -------------------------------------------------------------------- |
| 名字     | 攻击武技名                                                           |
| 描述     | 攻击武技描述                                                         |
| 修行公式 | 基于角色三维和武学素养的公式，结果为消耗一个行动点所能获得的修行经验 |
| 等级     | 当前等级                                                             |
| 稀有度   | 武技稀有度，分5级                                                    |
| 类型     | 字符串，可自定义                                                     |
| 境界     | 五个等级攻击武技的具体属性                                           |

攻击武技的境界结构如下：

| 属性     | 描述                       |
| -------- | -------------------------- |
| 等级     | 境界等级                   |
| 经验     | 升至当前等级需要的经验     |
| 武学素养 | 升至当前等级获得的武学素养 |
| 威能     | 当前等级武技的威能         |
| 蓄力时间 | 当前等级武技的蓄力时间     |
| 词条     | 当前等级武技具备的词条     | 


防御武技与攻击武技类似，差别仅在于威能变成守御，没有蓄力时间。
